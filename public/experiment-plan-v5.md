# INNO IROCA 測色実験計画 v5.0

**総サンプル数**: 272（測定: 各5回 → 合計1,360測定）

**ID形式**: `a01`, `f15`, `g27` — ファイル保存時は `a01-1`（ID＋測定回数）

---

## 優先順位と運用ルール

1. **① まず単品13薬剤** — 全ての土台。白髪100%/黒髪0% + 白髪率カーブ + アンカー
2. **② 次にFC混合レシピ** — メーカー公式FASHION COLOR表の32混合 × 白髪100%/黒髪100%
3. **③ 以降はモデル精度への寄与順** — 2色混色曲線 → 3色 → ブリーチ → GC → ブラインド
4. **同じ調合は1回だけ作る** — レシピでグルーピング済み。白髪/黒髪/各白髪率を一度に塗布
5. **Phase Eは最後に固定** — ブラインドhold-out、別バッチ
